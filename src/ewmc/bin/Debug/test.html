Welcome to the Extended Web Markup Language!

Extended Web Markup is a new language to describe websites, which compiles into HTML code.
The syntax looks like this:

head {
   title { "Test" }
   meta (author = "Test");
}

document {
   p {
      "Hello, World!"
   }
}

This page is written in EWML!
